# Dustie ☁️

Dustie is a quiet companion for unfinished thoughts.

Inspired by the spirit of the Korean classic chatbot "심심이", Dustie isn't designed to be a smart AI assistant. Instead, it encourages conversations that help unfinished thoughts, drafts, ideas, and curiosities stay alive.

> "흠냐... 궁금한데요 😄"

---

## Features

* 💬 Lightweight rule-based conversation
* 🧠 Context-aware reply system
* 📄 Draft generation from conversations
* ⬇️ Export drafts as TXT files
* 🎨 Custom Dustie UI theme
* 🌐 Flutter Web support
